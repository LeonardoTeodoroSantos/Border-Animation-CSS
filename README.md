# Border Animation CSS
O HTML aqui foi o mais básico possível, não era o foco do aprendizado. Relacionado ao CSS, aprendi sobre algumas coisas novas e coloquei em prática outros conhecimentos recém-adquiridos.

## Descrição
Um simples projeto, com a ideia de mostrar uma borda animada, que tem como objetivo trazer uma "animação" para os projetos. 

## Tecnicas e Tecnologias Utilizadas
![Static Badge](https://img.shields.io/badge/CSS-%23663399?style=for-the-badge&logo=CSS&logoColor=%23663399&labelColor=white&color=%23663399) ![Static Badge](https://img.shields.io/badge/HTML5-%23E34F26?style=for-the-badge&logo=HTML5&logoColor=%23E34F26&labelColor=white&color=%23E34F26)

- `HTML`: HTML básico, feito para organizar através da tag div.

- `CSS`: CSS também básico, aproveitei para usar coisas que estudei recentemente, e aprender coisas novas. Aperfeiçoar o conhecimento em relação aos `positions` e `z-index`. Aprendi propriedades novas, como o `content` vazio, com aspas simples utilizado nas pseudo-classes `before` e `after`, outra propriedade que utilizei pela primeira vez foi a `animation`. A propriedade `@keyframes` também é algo novo para mim, busquei na documentação <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/@keyframes">MDN</a>, serve para controlar uma animação, podendo controlar o começo, meio e fim, no meu caso usei pro inicio e fim, indicando que queria uma rotação de 360°.

## Funcionamento
A ideia do GIF é mostrar como funciona a borda animada, você pode determinar a largura que preferir, e também a velocidade com que a borda gira. A mudança de cor é determinada pela propriedade `background` escolhendo o linear-gradient e assim escolhendo as cores que deseja.
